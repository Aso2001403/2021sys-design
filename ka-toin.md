```uml
@startuml
 ユーザー -> webサーバー : カートの中クリック
 webサーバー -> ユーザー : カートの中表示
 


 alt 詳細へ
ユーザー -> webサーバー : 詳細へクリック
webサーバー -> DBサーバー : 商品名照会
DBサーバー -> DBサーバー : 商品名処理
DBサーバー -> webサーバー : 商品名結果
webサーバー -> ユーザー : 商品詳細を表示
 else 削除
ユーザー -> webサーバー : 削除クリック
webサーバー -> DBサーバー : 商品名照会
DBサーバー -> DBサーバー : 商品名処理
DBサーバー -> webサーバー : 商品名削除
webサーバー -> ユーザー : カート中を表示
 end


@enduml
```
