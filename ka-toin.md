```uml
@startuml
opt なし
 ユーザー -> webサーバー : カートの中クリック
 webサーバー -> DBサーバー : ユーザー登録
 DBサーバー -> DBサーバー : 登録処理
 DBサーバー -> webサーバー : 登録結果

 alt 詳細へ
ユーザー -> webサーバー : 商品名クリック
webサーバー -> DBサーバー : 商品名照会
DBサーバー -> DBサーバー : 商品名処理
DBサーバー -> webサーバー : 商品名結果
webサーバー -> ユーザー : 商品詳細を表示
 else 削除
 webサーバー -> ユーザー : 失敗メッセージを表示
 end
end

@enduml
```
