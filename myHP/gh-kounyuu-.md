```uml
@startuml
opt カテゴリ検索
ユーザー -> webサーバー :カテゴリ検索
webサーバー -> DBサーバー : カテゴリ照会
DBサーバー -> DBサーバー : カテゴリ処理
DBサーバー -> webサーバー : カテゴリ結果
webサーバー -> ユーザー : カテゴリを表示
ユーザー -> webサーバー : 商品名クリック
webサーバー -> DBサーバー : 商品名照会
DBサーバー -> DBサーバー : 商品名処理
DBサーバー -> webサーバー : 商品名結果
webサーバー -> ユーザー : 商品詳細を表示

alt カートに入れる
 ユーザー -> webサーバー :カートに入れるクリック
 webサーバー -> ユーザー:カートの中
 else 前の画面に戻る
 ユーザー -> webサーバー :前の画面に戻るクリック
 webサーバー -> ユーザー:商品一覧ページ表示
 end
end

ユーザー -> webサーバー : 商品名検索
webサーバー -> DBサーバー : 商品名照会
DBサーバー -> DBサーバー : 商品名処理
DBサーバー -> webサーバー : 商品名結果

alt 認証成功
webサーバー -> ユーザー : 商品名を表示
else 認証失敗
webサーバー -> ユーザー : 商品一覧を表示
end

@enduml
```
